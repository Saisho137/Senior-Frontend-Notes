# Senior Frontend Notes

Este repositorio centraliza notas sobre conceptos de alto nivel que suelen evaluarse en entrevistas para posiciones de **Senior Frontend Developer**.

## Contenido

### [15 Frontend Concepts](./15_Frontend_Concepts.md)

Notas basadas en el video: [15 Frontend Concepts Every Senior Dev Has Mastered](https://www.youtube.com/watch?v=tHbrjCtNx7Q)

Cubre conceptos fundamentales como:

- Critical Rendering Path
- Core Web Vitals
- HTTP Caching & Content Negotiation
- Lazy Loading & Bundle Splitting
- State Management Patterns
- Y más...

## Propósito

Este repositorio sirve como material de estudio y referencia rápida para preparación de entrevistas técnicas y reforzamiento de conceptos avanzados de desarrollo frontend. Cada archivo MD contiene notas organizadas sobre diferentes temarios y recursos relevantes para el rol de Senior Frontend Developer.
